# ✅ To-Do List (Interface de Tarefas)

Uma aplicação clássica de lista de tarefas (To-Do List) desenvolvida como atividade acadêmica. 

Este projeto é muito especial para mim, pois marcou um dos meus primeiros contatos práticos com a linguagem JavaScript, integrando-a com HTML e CSS para criar uma interface interativa.

**Tecnologias utilizadas:**
* HTML5
* CSS3
* JavaScript
